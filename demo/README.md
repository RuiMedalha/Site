# Demo - HotelEquip

Site de demonstração criado como showcase de design e UX.

## Como visualizar

**Opção 1 - Servidor Python (recomendado):**
```bash
cd demo
python3 -m http.server 8080
```
Depois abra: http://localhost:8080

**Opção 2 - Abrir diretamente:**
Abra o ficheiro `index.html` no browser (algumas funcionalidades podem não funcionar).

**Opção 3 - Node.js:**
```bash
npx serve demo
```
