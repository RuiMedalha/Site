# Hotelequip (novo tema WooCommerce)

Este repositório contém um **tema WordPress para WooCommerce** com design novo e mais clean.

## Arranque rápido (Docker)

```bash
docker compose up -d
```

Depois abre `http://localhost:8080`, instala WordPress, ativa o tema **Hotelequip Clean** e instala o plugin **WooCommerce**.
